Sistema Gardem

Entidad Descripción Función en el Sistema
Producto Representa un artículo individual en el catálogo de Gardem.
Ejemplo: Camiseta básica, Pantalón deportivo
 Base del catálogo Núcleo
Contiene información básica del artículo
Categoría Organiza los productos en grupos lógicos para facilitar la navegación.
Ejemplo: "Ropa Interior Hombre", "Deportiva Mujer"
 Clasificación Organización
Mejora la experiencia de búsqueda
Talla Define las opciones de tamaño disponibles para un producto específico.
Ejemplo: XS, S, M, L, XL, XXL
 Variante de producto Atributo
Permite personalización del pedido
Color Define las opciones de color disponibles para un producto específico.
Ejemplo: Azul marino, Rojo, Blanco, Negro
 Variante de producto Atributo
Amplía opciones de personalización
ItemCesta Representa un producto específico dentro de la cesta de pedido del cliente.
Ejemplo: 1x Camiseta Azul Talla M
 Elemento de compra Transacción
Vincula producto con cantidad y variantes
Cesta Agrupa los ItemCesta de un cliente antes de que se envíe el pedido.
Ejemplo: Carrito de compras temporal
 Contenedor de compra Proceso
Facilita la gestión pre-pedido
Administrador Representa al usuario de Gardem con permisos para gestionar el catálogo.
Ejemplo: Gestor de productos, Supervisor de ventas
 Usuario privilegiado Gestión
Control total del sistema
